Installation
-------------------
```
git clone https://github.com/Wykz/twitch-follow-bot

cd twitch-follow-bot

npm i
```

Now, put the bot token in ``index.js`` (line 243). Put your Twitch's tokens in ``tokens.txt`` and click on ``start.bat``.
